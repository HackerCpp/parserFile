Класс GeologySQLiteDB

Базовый класс для работы с базой данных SQlite3. Конструктор подключается к базе данных. Второй конструктор принимает открытую базу данных. Деструктор удаляет подключение если база создалась в конструкторе. Если конструктор получил базу деструкор ничего не делает. Для использования, нужно наследоваться от данного класса и использовать член класса m_db.Перед записью и чтением из/в базу, нужно проверять подключение (db->isOpen()). Подробнее...

#include <geologysqlitedb.h>
