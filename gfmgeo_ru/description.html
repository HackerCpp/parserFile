Геология gfmgeo

  GFMGEO (Геология) - комплексный программный продукт компании ООО"Геофизмаш" предназначенный для обработки геофизических каротажных данных после исследования скважин.Состоит из трёх программных модулей:

  1. Модуль данных
  2. Графический модуль
  3. Модуль обработки

Модуль данных в данном продукте позволяет открывать, конвертировать и сохранять различные форматы файлов, таких как .gfm .tlm .las и другие. Разработка шаблонов, отчётов, графических карт и моделей скважин возможна по требованию пользователей. Для облегчения работы разными пользователями есть возможность хранения информации в реляционной базе данных, которая может находиться на удалённом сервере.

Графический модуль позволяет визуально просматривать полученные данные в разных форматах. Возможны отрисовка линий, спектра, волновой картины, конструкций скважин и многое другое. Реализован удобный, интуитивно-понятный интерфейс взаимодействия пользователя для изменения характеристик просмотра данных.

Модуль обработки позволяет создавать новые данные или изменять уже имеющиеся. Новые объекты можно создавать в конструкторе объектов. Конструктор позволяет создавать все реализованные форматы такие как, спектр, волна, плотность, толщина и многие другие. Для гибкости обработки, сделана возможность изменения данных в скриптах интерпретатора python.

Все модули в целом образуют простой для пользователя ,но в тоже время очень гибкий инструмент для работы с каротажными данными. Возможно использование для простого редактирования, перед передачей в дальнейшую обработку,такого как: подвинуть по глубине, по времени, убрать лишние данные, добавить данные из другого файла.Также возможна полная обработка данных с применением сложных формул, сохранением отрисовок в разных форматах и генерации отчётов.
